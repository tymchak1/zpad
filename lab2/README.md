### Опис завдання

1. **VHI-індекс** (`part1_vhi.ipynb`) — аналіз Vegetation Health Index для областей України (дані NOAA).
2. **Household Power** (`part2_household.ipynb`) — аналіз споживання електроенергії (UCI ML Repository): фільтрація, нормалізація, кореляція, OHE.

### Інструкція із запуску

1. `pip install pandas numpy scipy scikit-learn`
2. Відкрити ноутбук → Run All (дані завантажуються автоматично)
