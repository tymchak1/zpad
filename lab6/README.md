### Опис завдання

Лабораторна робота №6 — C++ OpenCV. Програма читає відео з камери,
відображає у вікні, реагує на клавіші/мишу та вмикає режими обробки:
Normal, Invert, Blur, Canny, Sobel, Binary, Glitch, Draw.
Слайдер яскравості, обертання/зум стрілочками, малювання прямокутників мишкою, HUD з FPS.

### Інструкція із запуску

1. `./preinstall.sh` — встановлення залежностей (OpenCV, CMake, компілятор)
2. `./build.sh` — збірка проєкту
3. `./run.sh` — запуск

### Вимоги

- Linux (Ubuntu/Debian) або macOS
- Веб-камера
- CMake >= 3.10, g++/clang++ з підтримкою C++17
